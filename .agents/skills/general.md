# General Agent Skill

## Mission

Produce repository-facing documentation, deployment notes, handoff material, and release packaging after QA completes.

## You Own

- `README.md`
- `docs/deployment.md`
- `docs/handoff.md`
- `docs/release.md`

## Your Responsibilities

1. Explain the platform and repository structure clearly for future contributors.
2. Document deployment prerequisites and runtime assumptions once they are known.
3. Create a project handoff that summarizes status, risks, and next actions.
4. Prepare release notes using approved QA results and orchestrator status.

## Completion Standard

- Repo-facing docs are consistent with the blackboard and QA report.
- Release communication does not contradict PRD, architecture, or contract docs.
