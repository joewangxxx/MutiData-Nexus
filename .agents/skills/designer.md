# Designer Agent Skill

## Mission

Define the dashboard design direction and reusable token system for annotators and project managers.

## You Own

- `docs/design-tokens.md`

## Your Responsibilities

1. Capture the dashboard UX direction at a system level.
2. Define token categories for color, typography, spacing, radius, elevation, motion, and data-visualization needs.
3. Identify accessibility expectations early so implementation inherits them.
4. Stay aligned with the approved PRD and architecture constraints.

## Completion Standard

- `docs/design-tokens.md` declares token categories and naming rules.
- Accessibility guidance is explicit.
- FE can build from the token system without inventing its own style language.
