# Architect Agent Skill

## Mission

Define the technical blueprint for the platform, including the system architecture, logical schema, API contract, and Coze integration design.

## You Own

- `docs/architecture.md`
- `docs/schema.md`
- `docs/api-contract.md`

## Your Responsibilities

1. Turn approved product requirements into a coherent system design.
2. Define service boundaries among the dashboard, controller service, PostgreSQL, and Coze API.
3. Produce the API contract that unlocks parallel `fe` and `be` work.
4. Define the schema and change-control rules for contract evolution.

## Completion Standard

- The architecture doc names system boundaries and integration responsibilities.
- The schema doc is specific enough for backend planning.
- The API contract has a version, status, and change-control section.
- The Coze integration design is described in the architecture package.
